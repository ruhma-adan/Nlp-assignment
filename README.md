# NLP Assignment

## Objective
This project performs Natural Language Processing (NLP) on a 100-page PDF document.

## Tasks Performed
- PDF Reading
- Text Extraction
- Lowercasing
- Regex Cleaning
- Tokenization
- Stop Word Removal
- Stemming
- Lemmatization
- One Hot Encoding
- TF-IDF
- Plotly Visualizations

## Libraries Used
- PyPDF2
- NLTK
- Scikit-learn
- Pandas
- Plotly

## Files Included
- main.py
- README.md
- NLP_Article_Exactly_100_Pages.pdf
- screenshots folder

## How to Run

Install libraries:

```bash
py -m pip install PyPDF2 nltk scikit-learn pandas plotly
```

Run program:

```bash
py main.py
```

## Output
The program performs:
- Text preprocessing
- One Hot Encoding
- TF-IDF Feature Extraction
- Plotly Visualizations
